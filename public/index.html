UW Shibboleth SPA Multiserver with Nginx proxy

This is a SPA! Only this html file is served from the server, everything else is JSON. Visit the following routes in your browser, it's best to have a JSON viewer in your browser like Chrome's JSONView

  • Open.json...no security
  • Secure.json...UW Web Login with Shibboleth

Note: The Secure.json route will return a dummy user when running the site as localhost. See repo readme file for full details on how to set everything up.
